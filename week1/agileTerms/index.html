Planning Development in Agile
  1. Planning is adaptive.
  2. Planning is iterative.
  3. Iterations are defined by user stories. Planning with user stories. The main characters of user stories are actors. Take a minute with your team to identify the actors in your document.
  4. User stories are used to define the project requirements. Gathering requirements is a key part of the planning process.
  5. User stories have some value.
